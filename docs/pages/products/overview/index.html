Core Framing

ICOS is defined as a new category: a voice-trained reflective cognition system that organizes memory continuity, emotional patterning, profile intelligence, and long-term self-authored development into one governed product architecture.

The product layer is not a generic chatbot surface. It is a continuity system that helps a user structure thinking, preserve development across time, and turn reflection into a governed profile architecture.

This Overview page is the framing layer for the product itself. It explains what ICOS is, how it is positioned, and which internal dimensions belong to the category before deeper capability, workflow, integration, and software implementation layers are opened.
